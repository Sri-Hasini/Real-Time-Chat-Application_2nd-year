# 💬 Real-Time Chat Application

A full-stack **real-time chat application** that enables seamless communication with modern messaging features like instant messaging, typing indicators, media sharing, and audio/video calls.

---

## 🚀 Features

### 🔐 Authentication

- User Signup & Login
- Secure authentication system
- Session management

### 💬 Real-Time Messaging

- One-to-one chat
- Instant message delivery (no refresh required)
- Online/offline user status

### ✨ Advanced Chat Features

- Typing indicators
- Read receipts (if implemented)
- Message timestamps

### 📎 Media Sharing

- Send images, files, and documents
- Voice message recording & sharing

### 📞 Calling Features

- Audio calling
- Video calling

### 👥 Group Chat

- Create group conversations
- Real-time group messaging

---

### Frontend

### Backend

### Real-Time Communication

- Socket.IO

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Sri-Hasini/Real-Time-Chat-Application_2nd-year.git
cd Real-Time-Chat-Application_2nd-year
```

---

### 2️⃣ Install dependencies

#### Backend

```bash
cd backend
npm install
```

#### Frontend

```bash
cd frontend
npm install
```

---

### 3️⃣ Configure Environment Variables

Create a `.env` file in backend:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
JWT_SECRET=your_secret_key
```

---

### 4️⃣ Run the application

#### Start backend

```bash
npm run dev
```

#### Start frontend

```bash
cd frontend
npm run dev
```
